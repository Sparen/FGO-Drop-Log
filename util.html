GUDAGUDA Honnouji Shorthand Converter

Input: CSV string with units in the format X#x#
Output: Extended log-master object
EX: "H100x3,N3x1,C1x1,C2x3,H2x1" produces:
{ "uplog": false, "drop": [], "stackdrop": [{"id": "HONNOJI_PT", "stack": 100}, {"id": "HONNOJI_PT", "stack": 100}, {"id": "HONNOJI_PT", "stack": 100}, {"id": "T_NASU", "stack": 3}, {"id": "YT_CHAWAN", "stack": 1}, {"id": "YT_CHAWAN", "stack": 2}, {"id": "YT_CHAWAN", "stack": 2}, {"id": "YT_CHAWAN", "stack": 2}, {"id": "HIRA_GUMO", "stack": 2}] }
Please note that non-event items are not handled by this system and must be added to the drop array independently.
