QUERY ENGINE

This is the Query Engine for my database. Using this tool, the drop locations, rates, apd, and other stats for a given item can be dredged up.

I have chosen to exclude all EXP items from this list.

Events and quests that are no longer available (such as former dailies) are excluded from queries regarding standard items unless the first option below is selected. Hunting Quests are treated as Limited Time Quests and have a seperate option.

Event reruns will not be logged.
